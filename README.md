# heanbian-crypto

## 前提条件

JDK 21

## pom.xml

具体版本，可以从 Maven Central 获得

```
<dependency>
  <groupId>com.heanbian</groupId>
  <artifactId>heanbian-crypto</artifactId>
  <version>20.1</version>
</dependency>
```

## 使用示例


> AesTemplate

> RsaTemplate

> EcTemplate

> SM4Template


说明：支持ECC、RSA、AES、SM4。
